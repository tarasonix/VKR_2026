## Структура репозитория

| Файл | Описание |
|------|----------|
| `FunctionalUCB.ipynb` | Реализация алгоритма FunctionalUCB для многоруких бандитов с тяжёлыми хвостами распределений. Адаптивная оценка индекса хвоста Хилла, квантильная оптимизация. |
| `SCARDO.ipynb` | Эксперименты с вероятностной моделью SCARDO (стохастические переходы мнений по таблице влияния), сравнение бандит-алгоритмов для выбора собеседника. |
| `Модель_ДеГрута_м_Фридкина_Джонсена.ipynb` | Классические модели динамики мнений (Де Гроот, Фридкин–Джонсен) с использованием бандит-алгоритмов для выбора партнёров по взаимодействию. |
| `Контекстные_бандиты.ipynb` | Линейные многорукие бандиты (LinUCB, Linε-greedy, Linear Thompson Sampling) и их сравнение с неконтекстными аналогами. |
| `Динамика_мнений_с_контекстом_и_без.ipynb` | Основной эксперимент: мультиагентная модель на разреженных графах (Эрдёш–Реньи, Барабаши–Альберт, Ватс–Строгац) с 5 стратегиями обновления мнения. |

##  Реализованные алгоритмы

### Неконтекстные бандиты
- $\varepsilon$-greedy
- UCB (Upper Confidence Bound)
- Thompson Sampling

### Контекстные (линейные) бандиты
- Linear $\varepsilon$-greedy
- LinUCB (с Ridge-регуляризацией)
- Linear Thompson Sampling

### Специальные алгоритмы
- **FunctionalUCB** — адаптивный UCB для $q$-квантиля с оценкой индекса хвоста по Хиллу (для распределений с тяжёлыми хвостами)

##  Модели динамики мнений

1. **SCARDO** — вероятностная модель с дискретными мнениями и матрицами стохастических переходов, выведенными из данных ВКонтакте.
2. **Фридкин–Джонсен** — непрерывная модель с параметром упрямства $\alpha$.
3. **Де Гроот** — классическая модель взвешенного усреднения мнений соседей.


## Топологии социальных сетей

- **Эрдёш–Реньи (ER)** — случайный граф, пуассоновское распределение степеней.
- **Барабаши–Альберт (BA)** — безмасштабная сеть с предпочтительным присоединением (хабы).
- **Ватс–Строгац (WS)** — модель «малого мира» с высокой локальной кластеризацией и короткими глобальными путями.


## Используемые метрики

- **Накопленная награда** — сумма $r_i(t) = -|x_i(t+1) - \theta|$ по всем агентам и шагам.
- **Ошибка** — среднее абсолютное отклонение финального мнения от $\theta$.
- **Стандартная ошибка (SE)** — мера разброса финальных мнений между запусками.
- **Распределение стратегий** — частота выбора каждой из 5 стратегий обновления.
